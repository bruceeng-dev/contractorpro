## Job Management

1. **Basic Information**

#### Documents Tab
- Upload contracts, permits, and plans
- Organize by document type
- Version control and access tracking
- Share with clients and subcontractors

#### Estimates Tab
- Link related estimates to jobs
- Track estimate status and client responses
- Convert accepted estimates to active projects

## Smart Estimating System

### Template-Based Bidding
Speed up estimate creation with intelligent templates:

#### Pre-Built Templates
- Kitchen Renovations (cabinets, countertops, appliances)
- Bathroom Remodels (fixtures, tile, vanities)
- Roofing Projects (materials, labor, permits)
- Flooring Installation (materials, preparation, finishing)
- Exterior Painting (prep, materials, labor)

### Creating Professional Estimates

1. **Project Information**
   - Client details and project description
   - Auto-generated estimate number
   - Valid until date for quote expiration
   - Professional company branding

2. **Cost Breakdown**
   - **Labor Costs**: Hours × rates for different skill levels
   - **Material Costs**: Itemized materials with quantities
   - **Equipment**: Tool rentals and machinery
   - **Subcontractors**: Third-party services

3. **Automatic Calculations**
   - Subtotal of all direct costs
   - Overhead percentage (default 10%, customizable)
   - Profit margin (default 15%, adjustable)
   - Total project cost with all markups
   - Real-time updates as you edit line items

4. **Line Item Management**
   - Description and category for each item
   - Quantity and unit measurements (sqft, hours, each)
   - Unit cost and extended totals
   - Notes and specifications

### Estimate Workflow
1. **Create** - Use template or build from scratch
2. **Customize** - Adjust line items for specific project
3. **Review** - Check calculations and totals
4. **Send** - Email directly to client with professional formatting
5. **Track** - Monitor status (Draft, Sent, Viewed, Accepted, Rejected)
6. **Convert** - Transform accepted estimates into active jobs

## Troubleshooting & Support

### Common Issues

#### Photo Upload Problems
- **File Too Large**: Reduce image size (max 50MB)
- **Upload Fails**: Check internet connection and try again
- **Wrong Format**: Use JPG, PNG, or supported formats
- **Permissions Error**: Ensure uploads/ directory is writable

#### Email Notifications Not Sending
- **SMTP Settings**: Verify email configuration in .env file
- **Gmail Security**: Enable 2FA and use App Password
- **Firewall**: Check if SMTP ports are blocked
- **Recipient Issues**: Confirm client email addresses

#### Database Errors
- **Connection Failed**: Check database file permissions
- **Corruption**: Run `python migrate.py reset` to rebuild
- **Disk Space**: Ensure adequate storage available
- **Migration Issues**: Restart application after updates

#### Performance Issues
- **Slow Loading**: Clear browser cache and cookies
- **Large Images**: Compress photos before upload
- **Memory Usage**: Restart application if sluggish
- **Browser Compatibility**: Use Chrome, Firefox, or Safari

### System Maintenance

#### Regular Backups
- **Database**: Copy `contractorpro.db` file regularly
- **Uploads**: Backup `uploads/` directory
- **Configuration**: Save `.env` settings securely
- **Documentation**: Export important project data

#### Updates and Upgrades
- **Dependencies**: Run `pip install -r requirements.txt` for updates
- **Database**: Use migration scripts for schema changes
- **Features**: Check CLAUDE.md for new functionality
- **Security**: Update passwords and keys periodically

### Data Recovery

#### Lost Database
1. Restore from recent backup
2. If no backup, run `python migrate.py init`
3. Re-enter critical project information
4. Implement regular backup schedule

#### Corrupted Files
1. Check file permissions and ownership
2. Restore from backup if available
3. Use file recovery tools if necessary
4. Review system logs for error patterns
