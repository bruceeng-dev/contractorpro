# ContractorPro - Quick Start with Docker

## 🚀 Run in 3 Steps:

### 1. Install Docker Desktop
Download from: https://www.docker.com/products/docker-desktop

### 2. Open Terminal in this folder
- **Windows**: Right-click → "Open in Terminal"
- **Mac/Linux**: `cd` to this folder

### 3. Run this command:
```bash
docker-compose up -d
```

### 4. Open your browser:
```
http://localhost:5000
```

**Login:**
- Username: `admin`
- Password: `admin123`

---

## To Stop:
```bash
docker-compose down
```
